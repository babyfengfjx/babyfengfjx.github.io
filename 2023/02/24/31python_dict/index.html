setdefault()

根据给定的键获取相应的值。如果键不存在，则将其添加到字典中，并将值设置为指定的默认值（默认为None）

具体来说，setdefault() 方法会按照以下步骤操作：

  1. 如果 key 存在于字典中，则返回对应的值；
  2. 如果 key 不存在于字典中，则将 key 和默认值 default 加入到字典中，并返回 default。

如果我们再次调用 setdefault() 方法，并传入一个已经存在于字典中的键，例如 'apple'，那么该方法会返回该键对应的值，并不会改变字典的内容。

items()

返回字典中所有键值对的元组列表

keys()

返回字典中所有键的列表

pop()

根据给定的键删除并返回相应的值。如果键不存在，则返回指定的默认值（默认为None）

popitem()

随机删除并返回字典中的一对键值对。如果字典为空，则引发KeyError异常

update()

使用另一个字典中的键值对更新当前字典。如果键已经存在，则更新对应的值；否则，添加新的键值对

values()

返回字典中所有值的列表
